## 🚀🗼 CloudFlare Workers Example

**Installation**


```bash
git clone https://github.com/kardespro/cloudflare-workers
npm i wrangler wranglerjs
```

**Login Cloudflare Account**

```bash
wrangler login
```

**Publish Your App**

```bash
wrangler dev
wrangler publish
```
